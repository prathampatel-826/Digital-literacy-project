Project Description
The project was designed for the Digital literacy course and as a Student Digital Ambassador. Its purpose was to give students experience and to demonstrate how digital literacy is critical and will affect their future. The skills we have covered in this project and what I will be working on with this project include: how important it is to practice safe internet usage, how to communicate professionally, and how to establish a strong Digital Identity.

The project consists of five (5) different tasks, which include creating an Infographic, a Digital Portfolio, Code Practice, Professional Email Writing and Cyber Crime Awareness.
